\subsection{Question}

Show $ \sum_{k = 1}^{n} k^3 = \frac{ n^2 \left( n + 1 \right)^2 }{ 4 } $ by answering the following questions:

\begin{enumerate}
	\item Calculate $ k^4 - \left( k - 1 \right)^4 $
	\item Sum both sides from $ k = 1 $ to $ k = n $
	\item Then use telescoping and the formulas $ \sum_{k = 1}^{n}k = \frac{ n^2 + n }{ 2 } $ and $ \sum_{k = 1}^{n}k^2 = \frac{ 1 }{ 6 } n(n + 1)(2n + 1) $
\end{enumerate}

\subsection{Question}

Let $ \matr{A}, \matr{B} $ be $ n \times n $ matrix. Prove that $ \left( \matr{AB} \right)^{T} = \matr{B}^{T}\matr{A}^{T} $.
